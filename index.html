Welcome to IDT 650: Advanced Web Design

In this course you will be building an educational website while learning important aspects of web design. Each of these topics will be incorporated into your educational website. Your website will be hosted for free on Github, just like this one. After this class you will be able to include a link to your project in your portfolio.

For this course you should already have an understanding of HTML, CSS and JS and how each applies to web design. The prerequisite for this course is EDU 654.

Course Objectives

  1. Review/master basic web principles
  2. Utilize HTML, CSS, and JS to create interactive user experiences
  3. Execute key principles of web design and User Experience/User Interface
  4. Recognize the need for accessibility and inclusivity on the web
  5. Design web content for multiple viewing environments
  6. Interpret code from external sources
  7. Construct a multi-page website for a specific educational purpose and audience


About

The Masters of Science in Instructional Design and Technology online degree program prepares individuals to lead in the design, development, and integration of technology and online learning into academic and private sectors. The program focuses on the instructional design process, adult learning theory, communication, media, and technology.

The curriculum connects theory to practice, preparing educators and corporate trainers to design learning experiences that promote creativity and active learning within an online environment. It prepares professionals to critique the current educational technology innovations and related research and immerse themselves in learning engagements that promote global connections, instant feedback, and communication with remote audiences.
